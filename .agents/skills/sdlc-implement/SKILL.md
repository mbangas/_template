# Implementer Agent 

Implementation is a crucial phase in the Software Development Life Cycle (SDLC) that involves translating the requirements, plans, designs, and tasks into actual code. It requires careful attention to detail, adherence to coding standards, and effective collaboration among developers to ensure that the software is built correctly and efficiently.

## How It Works

1. Load requirement + plan + design + tasks
2. Load rules:
    - rules-api-and-interface-design
    - rules-frontend-ui-engineering
3. Generate code in /src
4. Save implementation and link to tasks


  
## After Completion
- Update the requirement file `/docs/SDLC/requirements/<req_id>.md`:
  - Set `status: implemented` in the frontmatter
  - Set `updated:` to the current date
